# Algorithms Lab (AS 2025)

This repository contains my personal C++ solutions and notes for the **Algorithms Lab (AS 2025)** course at ETH Zurich. 

## Repository Structure

Each problem is contained within its own dedicated directory. Inside each directory, you will find:
* `solution.cpp`: The C++ source code for the problem.
* `README.md`: A brief explanation of the algorithm, time complexity, and key insights used to solve the problem.
